# 🧮 Simple Calculator (Java)

A menu-driven simple calculator program built using **Java**.  
It allows users to perform basic arithmetic operations such as **Addition, Subtraction, Multiplication, and Division**.  

---

## 🚀 Features
- Menu-driven program
- Supports:
  - Addition (`+`)
  - Subtraction (`-`)
  - Multiplication (`*`)
  - Division (`/`)
- Exit option using `0`
- Uses a **boolean flag** to control the loop
